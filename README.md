
# Go Clothing

This was a collaborative project made for the Alkemy Acceleration program.



## Tech Stack

**Client:** React, Redux, CSS, Formik, Yup

**Server:** Node, Express, MongoDB

## Features

- Login & Register
- Admin dashboard
- Create products
- Delete Products
- Update Products




## Installation

Install my-project with yarn

```bash
  yarn install 
```
